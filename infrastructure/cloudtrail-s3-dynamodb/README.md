## Instructions to use terraform

## What is needed

2 Files - providers.tf (telling terraform which provider it will use) and main.tf (our main script, what will we do)

## How to run

### First:
```
terraform init
```

### Second:
```
terraform plan
```
This will show what resources will be affected and if you have any issues with you code.

```
terraform apply
```
If we sure everything is on it's place we use this command to run our code.
