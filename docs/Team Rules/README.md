## AnomAI Team Rules

### Cost
AWS is cloud environment that has some free some paid services. The budget alert set to not exceed 15$ a month but we should try to make it cost-effective as we can so, be careful while using some services, be awere of the price.

### Tags
When you create resouces in AWS make sure to name it properly so we don't get lost, try to have tags and descriptions for the resources. Using tags is the best, you can create tags like Owner: Aykhan so we know who done what.

### File Overwriting
Using github, github codespace can be dangerous in order to not harm to each other let's work in seperate folders and not edit each other's files.
